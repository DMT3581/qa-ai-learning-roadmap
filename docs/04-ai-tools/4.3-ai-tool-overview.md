# AI Tool Overview: ChatGPT, Claude, Gemini and GitHub Copilot

> **⚠ Snapshot — verify before relying on specifics.** This space changes monthly: model names, versions, and features below are a snapshot as of **June 2026**. Treat the *positioning* of each tool as durable and the *version numbers as perishable* — re-check the official pages (linked at the end) before quoting specifics.

## In one line

The big general assistants — ChatGPT, Claude, and Gemini — have largely converged on a similar core, so the practical differences are now about ecosystem and where each is strongest; GitHub Copilot is the odd one out, a coding tool that lives in your editor and can even run the others.

## Why this matters for QA

You'll be offered several tools and need a quick sense of what each is *for* without getting lost in benchmark wars. By mid-2026 the three general assistants can all reason well, handle images, search the web, run code, and act somewhat agentically — so "which is best" is usually the wrong question. The better question (from *How to Choose the Right AI Tool*) is which one is approved at your firm, fits your workflow, and is strongest for the task in front of you.

## Core content

### ChatGPT (OpenAI)

The most widely used general assistant. As of mid-2026 it runs the **GPT-5 family**, with a default model that answers quickly and a router that escalates to a deeper "thinking" mode for harder problems. It's a strong all-rounder for writing, analysis, coding, and multimodal work, with web search and memory of past chats, plus a coding-focused mode (Codex) for agentic development. **For QA:** a solid default for brainstorming, test design, and analysis.

### Claude (Anthropic)

A general assistant known for strong coding, long-horizon agentic work, careful instruction-following, and handling long documents, with a pronounced safety and reliability focus. The lineup is tiered — **Opus** (maximum capability; *Opus 4.8* is the current flagship), **Sonnet** (the balanced default), and **Haiku** (fast and cheap) — with a newer top tier above Opus (the *Mythos*-class *Fable 5 / Mythos 5*) whose access is currently restricted. It includes Claude Code (an agentic coding tool) and Artifacts, and a large context window. **For QA:** strong for test design, reviewing code or test artifacts, and analysing long specs.

### Gemini (Google)

A general assistant with the strongest multimodal story — text, images, audio, and video — plus a very large context window, image and video generation, and deep integration across the Google ecosystem (Workspace, Search, Chrome, Android). The current **Gemini 3 family** offers a Pro tier for reasoning and coding, a fast Flash tier, and a heavier "Deep Think" reasoning mode. **For QA:** valuable when the task is multimodal (a screenshot of a failing UI, a screen recording) or lives inside Google Workspace.

### GitHub Copilot

Not a general chatbot but the **developer / software-lifecycle tool**. It lives in your editor (VS Code, JetBrains, Visual Studio) and on GitHub, offering inline completion, chat, an **agent mode** that edits across files and runs commands, a cloud **coding agent** that turns an issue into a pull request, a CLI, code review, and MCP support. Crucially, it's largely **model-agnostic**: a model picker lets you run GPT, Claude, or Gemini under the hood. **For QA (especially automation):** writing and refactoring test code in-repo, generating tests beside your code, and CI-related tasks — squarely your Layer 2 territory.

### At a glance

| Tool | What it is | Where it shines | Typical QA use |
|---|---|---|---|
| **ChatGPT** | General assistant | Versatile all-rounder, web + memory | Brainstorming, test design, analysis |
| **Claude** | General assistant | Coding, long docs, careful/safe output | Test design, code/artifact review, long specs |
| **Gemini** | General assistant | Multimodal, huge context, Google integration | Screenshots/recordings, Workspace tasks |
| **GitHub Copilot** | Coding tool (in IDE/GitHub) | In-editor coding, agentic dev, model choice | Writing/refactoring test code, CI tasks |

### How to read this

The three general assistants overlap so much that there's rarely a single "best" — pick by what's **approved** at your firm, where you already **work**, and which is strongest for the **task**. Copilot is complementary: the in-editor coding companion that can itself call the others. Whichever you use, use the **approved enterprise tier** for any work content (see *Confidentiality and Sensitive Data*), and re-verify versions because they change fast.

## Common pitfalls

- **Treating version numbers and benchmarks as durable.** They change monthly.
- **Assuming one tool is universally best.** The general assistants are close; fit decides.
- **Picking by hype rather than what's approved or where you work.**
- **Using a consumer tier for work data.** Use the approved enterprise tier.
- **Forgetting Copilot can run the other models.** "Copilot vs. Claude" is often a false choice.

## Key takeaways

- ChatGPT, Claude, and Gemini are converging general assistants; choose by ecosystem and fit, not a single "best."
- Claude leans coding/long-docs/safety; Gemini leans multimodal/Google; ChatGPT is the versatile default.
- GitHub Copilot is the in-editor coding tool, and is largely model-agnostic.
- Specifics here are a June 2026 snapshot — verify current versions before relying on them.
